# 📅 IVP・AWS 統合スケジュールカレンダー

AWARENESS（AWS）と IVP（WEB会員専用）のスケジュールを同期・統合して表示するWebアプリです。

## 🌐 アプリを開く

**👉 [統合カレンダーを開く](https://ivp-aws-calendar-ul64f56aegblbpykmr279z.streamlit.app)**

## 主な機能

- ☁️ AWSセミナー情報の自動取得（5ヶ月分）
- 🍊 IVP会員スケジュールのログイン連携（5ヶ月分）
- 📅 FullCalendar.js による月間カレンダー表示
- 🔍 一覧検索・CSVダウンロード
- 🚀 近日開催イベントのハイライト表示
